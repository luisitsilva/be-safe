
🛡️ BE-SAFE CyberDefense Matrix

BE-SAFE is an interactive security defense tool for Ubuntu, combining rootkit scanning, malware detection, intrusion prevention, and system 
auditing into a single command-line interface.

Version: 2.0 – “BE-SAFE tool”
Author: luisitsilva@flashintell.net

Target: Ubuntu 25.10 "Questing Quokka"



⚡ Features

Full Spectrum Scan – Deep system scan using rkhunter, chkrootkit, ClamAV, and Lynis
Rapid Response Scan – Quick threat detection
Deep File Integrity Scan – Tripwire-style file checks
Real-Time Security Dashboard – CPU, memory, SSH attempts, open ports, Fail2ban status
Live Log Feed – Tail multiple logs with color-coded alerts
Fail2ban Management – View jailed IPs, unban interactively
